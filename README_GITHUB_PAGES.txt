
ShelfAssured — GitHub Pages Deploy

1) Create a new repo on GitHub (e.g., shelfassured-demo).
2) Unzip this package and upload ALL files (index.html, images, .nojekyll) to the repo root.
3) In the repo: Settings → Pages → Source: "Deploy from a branch" → Branch: main • Folder: / (root) → Save.
4) Your site will publish at https://<your-username>.github.io/<repo-name>/
5) Share the link! (Cmd/Ctrl+Shift+R for hard refresh if you update files.)

Optional:
- Add collaborators under Repo → Settings → Collaborators
- To use a custom domain later, set it in Settings → Pages and add a DNS CNAME.
